# Terminus — AI Logo Generation Brief

---

## What Terminus Is

A ruthless backtesting engine for trading strategies. Not a friendly tool — a judgment machine. Every strategy that enters gets tested across 8 years of real market data, year by year, with frozen parameters and no cheating. It either survives or it doesn't.

**Name:** Terminus  
**Subtitle:** End of All Trades  
**Tagline:** *Where strategies go to prove themselves — or die.*

---

## Core Symbolism

### Primary metaphor: The Last Station
Terminus in Roman mythology = the god of boundaries and final markers. In transit = the last stop on the line. The logo should feel like something has reached its final destination — no further, this is where it ends.

### Secondary metaphor: The Gauntlet / Forge
Related to the original internal name (Crucible). Extreme conditions reveal truth. Metal either hardens or shatters. This is not a friendly incubator — it's a filter.

### What the logo should communicate:
- **Finality** — this is the end, not the beginning
- **Rigor** — no shortcuts, no softness, no false positives
- **Precision** — clean lines, calculated, not chaotic
- **Weight** — decisions made here carry consequence
- **Controlled austerity** — not aggressive or violent, but uncompromising

---

## Visual Direction

### Mood
- Industrial precision
- Roman/classical weight meets modern terminal aesthetic
- Dark, deliberate, minimal
- Not "crypto bro neon" — sophisticated and quiet
- Think: CFA exam manual meets UNIX terminal

### Style reference points
- Roman column capital (terminus stones were boundary markers, often column-like)
- Railway end-of-line buffer stop / terminus sign
- Old astronomical charts — precision, measurement, finality
- UNIX/CLI aesthetic — monospace, command line, `>` prompt
- Scales of justice — but not naively depicted, abstracted into geometry

## Color Palette

### Primary (preferred direction: dark)
| Role | Color | Hex | Notes |
|------|-------|-----|-------|
| Background | Near-black | `#0D0D0F` | Deepest dark, almost no blue |
| Primary mark | Off-white / bone | `#E8E4DC` | Warm white, aged, not pure |
| Accent | Muted amber | `#C8922A` | Like old gold, not neon |
| Secondary | Deep slate | `#2C2F3A` | For depth / shadow |

### Alternate direction: Light / print-friendly
| Role | Color | Hex |
|------|-------|-----|
| Background | Warm parchment | `#F5F0E8` |
| Primary mark | Charcoal | `#1A1A1F` |
| Accent | Bronze | `#8B6914` |

### Avoid
- Bright green (too "go signal" — this tool kills strategies)
- Red (too aggressive, implies failure not judgment)
- Blue gradients (crypto/tech cliché)
- Purple (fintech SaaS cliché)
- Any neon or glow effects

---

## Typography Direction

### Wordmark style
- **Terminus** in a geometric slab serif or Roman-influenced display type
- Weight: medium to bold — confident but not shouting
- Letter spacing: slightly wide — measured, deliberate
- No italics (italics feel like movement; Terminus is a stop)
- All-caps or mixed case both work; all-caps feels more monumental

### Reference typeface families (don't use exact — for aesthetic direction only)
- Freight Display (editorial gravitas)
- Canela (classical warmth)
- Playfair Display (Roman structure)
- DM Serif Display (clean, legible, weight)
- Or a geometric monospace if leaning CLI aesthetic (like `Fira Code` vibes in display size)

### "End of All Trades" subtitle
- Smaller, spaced out, lighter weight
- Could be small caps
- Should feel like an inscription, not a tagline

## Mark / Icon Concepts

### Concept A — The Terminus Stone
Roman terminus stones (termini) were upright stone markers at the edge of territory. A simplified silhouette:
- Rounded-top rectangular stone / stele
- Could have a subtle "T" carved into it
- Feels: ancient, boundary, final marker
- Risk: looks too tombstone-y without careful handling

### Concept B — End of Track / Buffer Stop
Railway buffer stop — the red-and-white striped barrier at the end of the line:
- Highly stylized, geometric abstraction
- Two horizontal bars with a vertical stop
- Could abstract into a "T" shape naturally
- Feels: engineering, finality, literal "last stop"
- Works well as an icon mark without text

### Concept C — The Scale Needle at Zero
A balance scale needle pointing straight down — perfectly balanced, perfectly judged:
- Clean geometric: a vertical line with a horizontal crossbar (naturally a "T")
- Suggests: the verdict is in, measured and final
- Very minimal — single color works at any size

### Concept D — Terminal Cursor / Command Prompt
`terminus>` or `_` cursor — the tool lives in the terminal, runs via CLI:
- A blinking cursor (rectangle) next to the wordmark
- Monospace feel
- Feels: developer tool, honest, no UI theater
- Works as wordmark, not standalone icon

### Concept E — Abstract "T" as Column Capital
A Roman column capital viewed front-on abstracts naturally into a "T":
- Wide horizontal slab on top, vertical shaft below
- Weight-bearing, structural, final
- Feels: classical authority, Latin origin of the name
- Very clean as a geometric mark

## What the Logo Must Work For

| Use case | Requirements |
|----------|-------------|
| GitHub repo header | Works on white background, readable small |
| PyPI package page | Works at 64×64px and 128×128px |
| CLI splash screen | Works in terminal (ASCII or small bitmap) |
| README badge | Works as favicon / 16px |
| Social media (Twitter/X, LinkedIn) | Square format, high contrast |
| Dark mode README | Must work on `#0D0D0F` background |
| Sticker / physical | Single color version must exist |

**Required formats from any AI generation:**
- Full color version (dark background preferred)
- Light version (for white backgrounds)
- Single color / monochrome version
- Square icon mark without wordmark
- Horizontal lockup (icon + "Terminus" text)

---

## What to Avoid

| Avoid | Why |
|-------|-----|
| Charts / candlesticks in the logo | Looks like every other fintech tool |
| Bull/bear animal imagery | Crypto cliché |
| Lightning bolts / speed imagery | Terminus is about judgment, not speed |
| Upward arrows | Contradicts the "end" theme |
| Glowing effects / gradients | Too modern/soft for the aesthetic |
| Thin hairline weights | Won't survive favicon size |
| Complexity with >3 elements | Must reduce to single color cleanly |
